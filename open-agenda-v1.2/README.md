# Open-Agenda v1.2 🔥

**Firefox-Style Modern Calendar UI**

---

## 🌟 What's New in v1.2

### 🔥 Firefox-Style Modern UI
- Tab-based navigation (Calendar, Events, Dashboard, Settings)
- Firefox-inspired color palette (orange accent)
- Rounded corners and modern cards
- Sidebar with icon navigation
- Global search bar

### 🎯 Features
- 📅 Calendar View with month navigation
- 📋 Events List View
- 📊 Dashboard with statistics
- ⚙️ Settings panel
- 🌙 Dark/Light theme toggle
- 🔍 Global search
- 💾 Auto-save
- 📤 Export (JSON, CSV)
- 📥 Import (JSON)

---

## 📦 Installation

```bash
sudo dpkg -i open-agenda_1.2.0_all.deb
sudo apt-get install -f
```

Or run directly:
```bash
python3 open-agenda-v1.2.py
```

---

## ⌨️ Keyboard Shortcuts

- `Ctrl+N` - New event
- `Ctrl+F` - Search
- `Ctrl+S` - Save
- `Ctrl+T` - Go to today
