# Open-Agenda v1.1 🗓️

**The Ultimate Smart Calendar** - Open-Agenda v1.1 - Ultra Modern 2025 Interface!

---

## 🌟 What's New in v1.1

### 🚀 Ultra Modern 2025 Interface
- Beautiful dark theme with purple accents
- Modern color palette and gradients
- Sleek card-based layout
- Improved visual hierarchy

### 💾 Auto-Save
- Your data is automatically saved every 1.5 seconds!
- No more lost data
- Visual indicator shows save status

### 📅 Multiple Calendars
- Personal, Work, Family, Health calendars
- Create custom calendars
- Toggle visibility per calendar
- Color-coded events

### 🎨 Event Categories
- Personal (Pink), Work (Blue), Family (Green)
- Health (Amber), Education (Purple), Social (Cyan)
- Finance (Red), Travel (Orange), Other (Gray)

### ⚡ Priority Levels
- High (Red), Medium (Amber), Low (Green), None (Gray)

### 🔍 Search
- Search across all events
- Search by title, location, or notes

### 📤 Export Options
- JSON (full data)
- CSV (spreadsheet compatible)
- ICS (iCalendar for other apps)

### 📊 Statistics Dashboard
- Total events count
- Events per category
- Events per priority
- Days with events

### ⌨️ Keyboard Shortcuts
- `Ctrl+N` - New event
- `Ctrl+F` - Search
- `Ctrl+S` - Save
- `Ctrl+T` - Go to today
- `Ctrl+E` - Export
- `Left/Right` - Navigate months

### 🔄 Recurring Events
- Daily, Weekly, Monthly, Yearly
- Auto-generate future events

### 📍 Additional Fields
- Location (optional)
- Notes (optional)
- Reminder toggle

### 🌗 Light/Dark Theme
- Toggle between themes
- Persistent preference

## 📦 Installation

```bash
sudo dpkg -i open-agenda_1.1.0_all.deb
sudo apt-get install -f
```

Or run directly:
```bash
python3 open-agenda-v1.1.py
```

---

## 📖 Usage

1. **Add Events**: Fill in the form and click "Add Event"
2. **Manage Calendars**: Use sidebar to toggle calendar visibility
3. **Search**: Use the search bar at the top
4. **Export**: Click "Export" button for export options
5. **Statistics**: Click "Stats" button to view dashboard
